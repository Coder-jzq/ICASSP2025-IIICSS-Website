Intra- and Inter-modal Context Interaction Modeling for Conversational Speech Synthesis
ABSTRACT
Conversational Speech Synthesis (CSS) aims to effectively use dialogue history to infer and generate speech with appropriate conversational prosody. Multimodal dialogue history (MDH) includes text and audio modalities. Previous CSS work typically models the entire MDH implicitly to obtain conversational prosody. However, modeling the intra-modal and inter-modal interactions in MDH can more effectively capture subtle semantic and prosodic variations, enabling more accurate inference of the target utterance's conversational prosody. Previous work ignores this point. Therefore, we propose a novel Intra-modal and Inter-modal Context Interaction Modeling CSS model, termed I³-CSS, which includes: 1) Intra-Modal and Inter-Modal Interaction Modules: Using a contrastive learning-based interaction enhancement method to model intra-modal and inter-modal interactions within MDH to infer the semantic and prosodic of the target utterance. 2) Speech Synthesizer: Integrates the inferred semantics and prosody to generate speech with appropriate conversational prosody. Experiments show that I³-CSS can synthesize more suitable conversational prosody compared to baseline models. Code and audio samples are available at https://github.com/Coder-jzq/ICASSP2025-IIICSS.

EXPERIMENTS
Comparative Experiment:
1) DailyTalk is an advanced CTTS baseline. It proposes coarse-grained text context modeling in dialogue history to enhance speech expressiveness.
2) M²-CTTS designs a textual context module and an acoustic context module with both coarse-grained and fine-grained modeling, generating speech with better prosody and naturalness.
3) CONCSS incorporates a contrastive learning module to ensure the discriminability of contextual features during dialogue history modeling.
4) Homogeneous Graph-based CSS adopts the homogeneous graph to model the multi-modal context in conversation, inferring both the global and local speaking styles of the current utterance.
5) ECSS is an emotional expressive conversational speech synthesis model. It constructs multi-source knowledge into a heterogeneous graph and incorporates an emotion prediction task to synthesize emotionally expressive speech.
